HTML es el lenguaje básico para crear páginas web, ya que organiza el contenido mediante etiquetas como textos, imágenes y enlaces.

Etiquetas de Formato

Las etiquetas `hr` y `br` en HTML se utilizan para organizar el contenido de una página web. La etiqueta `hr` sirve para insertar una línea horizontal que separa secciones o temas, mientras que `br` se usa para hacer un salto de línea dentro de un texto sin iniciar un nuevo párrafo.
