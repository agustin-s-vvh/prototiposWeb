Introducción al lenguaje

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Permite organizar el contenido mediante etiquetas, como títulos, párrafos, imágenes y enlaces, para que los navegadores puedan interpretarlo y mostrarlo correctamente.
No es un lenguaje de programación, sino de marcado, ya que define la estructura y el significado del contenido en internet.

Etiquetas de Formato

Las etiquetas 'hr' y 'br' en HTML sirven para organizar el contenido visual de una página. La etiqueta 'hr' se utiliza para insertar una línea horizontal que separa secciones o temas dentro del contenido, ayudando a estructurarlo mejor.
Por otro lado, la etiqueta 'br' se usa para hacer un salto de línea dentro de un texto, sin crear un nuevo párrafo, permitiendo que el contenido continúe en la siguiente línea.
